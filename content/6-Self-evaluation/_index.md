### Overview

During my 12-week internship as an FCJ Cloud Intern at **Amazon Web Services Vietnam Co., Ltd.** (from April 17, 2026, to July 30, 2026), I had the incredible opportunity to transition from learning foundational cloud computing concepts to designing and deploying enterprise-grade Serverless architectures. 

This internship was not just about theoretical knowledge; it was a hands-on journey where I applied everything I learned into building a robust, fully automated, and highly secure **Serverless Helpdesk Portal**.

### Technical Skills Gained

* **Cloud Infrastructure & Networking:** Gained deep understanding of configuring VPCs, Subnets, Security Groups, and managing compute and database resources using Amazon EC2, EBS, and RDS.
* **High Availability & Disaster Recovery:** Successfully implemented AWS Backup plans and Auto Scaling Groups (including Predictive Scaling) to ensure applications are resilient, highly available, and cost-optimized.
* **Serverless Computing:** Mastered the Serverless stack by utilizing Amazon API Gateway, AWS Lambda, and Amazon DynamoDB. I learned how to decouple systems using asynchronous event-driven patterns with Amazon SQS (FIFO & DLQ) and Amazon SES.
* **DevOps & CI/CD:** Developed a strong DevOps mindset by automating the entire software release lifecycle using AWS CodeCommit, CodeBuild, CodeDeploy, and CodePipeline.
* **Security & Monitoring:** Learned to implement multi-layered security using Amazon Cognito (JWT), AWS WAF, and AWS KMS encryption. I also configured proactive monitoring and alerting using Amazon CloudWatch and Amazon SNS.

### Soft Skills & Personal Growth

* **Problem Solving & Troubleshooting:** Debugging complex CloudFormation templates, resolving Load Balancer health check failures, and tracing Lambda executions via CloudWatch Logs significantly improved my analytical thinking and troubleshooting speed.
* **Architecture Design:** I learned to "think like a Solutions Architect." Instead of just writing code, I now consider the Well-Architected Framework—evaluating security, reliability, performance, and cost before building.
* **Time Management & Documentation:** Keeping a detailed weekly worklog and adhering to strict project deadlines taught me the importance of clear documentation and consistent time management.

### Final Thoughts & Future Direction

The climax of this internship was building the **Serverless Helpdesk Portal**. Seeing a project grow from an empty VPC to a fully automated CI/CD pipeline deploying a secure, event-driven application was incredibly rewarding.

This internship has solidified my passion for Cloud Computing and DevOps. Moving forward, I plan to dive deeper into containerization (EKS) and Infrastructure as Code (Terraform) while aiming for my AWS Certified Solutions Architect credential. I am deeply grateful to the mentors at FCJ and AWS Vietnam for their guidance throughout this journey.
